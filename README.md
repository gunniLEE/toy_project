- 모델 파이프라인 구축을 위한 툴로 kubeflow 를 활용한다.
- 모델 형상관리와 모델서빙을 위한 툴로 MLflow를 활용한다.
- 데이터셋 형상관리를 위한 툴로 minio를 활용한다.
- 검색엔진을 위한 툴로 ElasticSearch를 활용한다.
